# Random Users

A simple React app that fetches random user data from the FreeAPI public API and displays each user inside a clean card layout.

## Features

- Fetches random users from an API
- Shows each user's name, email, and age
- Displays loading and error messages
- Uses a responsive card grid
- Includes clean custom CSS styling

## Tech Stack

- React
- Vite
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Run linting:

```bash
npm run lint
```

## API

The app uses this endpoint:

```text
https://api.freeapi.app/api/v1/public/randomusers
```

The fetched data is rendered as user cards on the page.
